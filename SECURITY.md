# Security Policy

## Reporting a Vulnerability
Please do not open public issues for security vulnerabilities.
Report vulnerabilities privately to the project maintainer and include:
- Affected component and version
- Reproduction steps
- Potential impact

## Scope
This project processes sensitive tax and identity information. Treat all data as confidential.

## Best Practices
- Never commit W-2 files, SSNs, or other PII.
- Prefer local model execution for sensitive documents.
- Redact logs and examples before sharing.
