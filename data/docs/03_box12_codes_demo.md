# Box 12 Codes (Demo Quick Notes)

Selected demo examples:

- Code D: Elective deferrals under a 401(k) plan.
- Code DD: Cost of employer-sponsored health coverage (reporting only).
- Code W: Employer and employee contributions to an HSA.

Interpretation tips:
- Presence of a code does not automatically change tax due in the same way for every filer.
- Use code descriptions as context and confirm treatment in trusted tax guidance.
